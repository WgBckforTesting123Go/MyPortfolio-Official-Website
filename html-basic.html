HTML အခြေခံ တည်ဆောက်ပုံ

တင်သည့်ရက် - ဧပြီ ၂၆၊ ၂၀၂၆ | ဖတ်ရှုချိန် - ၅ မိနစ်

HTML ဆိုတာ **HyperText Markup Language** ရဲ့ အတိုကောက်ဖြစ်ပြီး ဝဘ်ဆိုဒ်တစ်ခုရဲ့ အရိုးအဆောက်အအုံကို တည်ဆောက်တဲ့ နေရာမှာ သုံးပါတယ်။

HTML ရဲ့ အခြေခံ Code ပုံစံ

အောက်မှာ ပြထားတာကတော့ HTML ဖိုင်တစ်ခုမှာ မဖြစ်မနေ ပါဝင်ရမယ့် Structure ပါပဲ-

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>Hello World!</h1>
  </body>
</html>

သိထားသင့်သော Tags များ

  • <h1> to <h6>: ခေါင်းစဉ်များအတွက် သုံးသည်။
  • <p>: စာပိုဒ် (Paragraph) များအတွက် သုံးသည်။
  • <a>: Link များ ချိတ်ရန် သုံးသည်။
မှတ်ချက်: HTML Tags တွေကို အဖွင့်ရှိရင် အပိတ်ပြန်လုပ်ဖို့ မမေ့ပါနဲ့။
